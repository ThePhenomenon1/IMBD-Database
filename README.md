# IMBD-Database
Creating your own dataset. Scraping multiple pages of data from the IMDB website, in a single script, to fetch top 100 movies metadata. (Using Beautiful Soup and requests.)

Especially the metadata:
    
# 1. Movie Name 
# 2. Release Year
# 3. Watch Time 
# 4. IMDb Rating
# 5. Metascore
# 6. Votes
# 7. Gross Collection
# 8. Description
